# Neural Network Digit Recognizer

## 📌 Description
This project is a Neural Network Digit Recognizer built using pure HTML, CSS, and JavaScript. 
It allows users to draw digits (0–9) on a canvas and predicts the number in real-time using a 3-layer neural network (MLP).

## 🧠 How It Works
- Input image is converted to 28×28 pixels (784 inputs)
- Passed through 3 layers: 784 → 128 → 64 → 10
- Uses ReLU activation and Softmax for output
- Displays predicted digit with confidence score

## 🚀 Features / Advantages
- Real-time digit recognition
- No frameworks or libraries used
- Runs fully in the browser (no backend needed)
- Interactive UI with drawing canvas
- Visual neural network representation

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)

## ▶️ How to Use
1. Copy the code into a file named `index.html`
2. Open the file in any browser (Chrome recommended)
3. Draw a digit (0–9) on the canvas
4. Click **Predict**
5. View the predicted digit and confidence score

## 📌 Note
No installation, no server, and no internet required (except for initial font loading).
